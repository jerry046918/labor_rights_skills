# 争议类型识别与处理目录

> 本文件供 SKILL.md 在阶段 2（动态深化）加载使用。
> 每个争议类型包含 5 个字段：识别关键词 / 必问项 / 加分项 / 证据清单 / 法条映射。

## 使用说明

**加载时机：** SKILL.md 在阶段 1（基础信息收集）完成后、阶段 2（动态深化）开始时加载本文件。

**字段含义：**
- **识别关键词**：用户描述中触发该争议类型判定的关键词（含同义词、口语化表达）。命中后即视为该类型"激活"，但 SKILL.md 应对多个类型同时评估，单一用户案件可同时落入多个类型。
- **必问项**：5-8 个必问字段，每项标注"用于..."说明该字段在后续证据链构建或法条适用中的具体作用。SKILL.md 必须逐项询问，不得跳过。
- **加分项**：3-5 个可选补充信息，命中后能强化主张或扩大赔偿，但缺失不影响基本诉求构成。
- **证据清单**：按"必备 / 重要 / 加分"三级分级。必备与重要项标注"证明目的"与"固定建议"（固定操作指引）；加分项仅标注"证明目的"。
- **法条映射**：主要依据（劳动合同法/劳动法具体条款）+ 配套法规 + 地方性规定。标注"需 WebSearch 验证最新版"的为非主法、需要 SKILL.md 在生成最终意见时二次校验。

**注意事项：**
1. 同一案件常涉及多个争议类型（例如"违法解除"+"拖欠工资"+"加班费"+"经济补偿"），SKILL.md 应并行激活。
2. 必问项中的日期、金额、人数等量化信息必须追问到具体数值，不接受"大概"、"很多"等模糊表述。
3. 法条引用以 `references/labor-contract-law-full.md`（劳动合同法，98 条）和 `references/labor-law-full.md`（劳动法，107 条）为基准；其他法规仅作指引。

## 类型 1：违法解除 / 违法辞退

### 识别关键词
- "被辞退"、"被开除"、"被解雇"、"解除劳动合同"、"终止劳动合同"
- "通知我不用来了"、"让我办离职"、"被劝退"、"被要求自动离职"
- "说我违纪"、"说我能力不行"、"说我业绩不达标"、"说我考核不合格"
- "赔了 N+1"、"给了 1 个月工资"、"赔了 2N"
- "没提前 30 天通知"、"口头通知"、"电话通知"
- "辞退理由写的是…"、"解除原因是…"
- 处于孕期 / 产期 / 哺乳期 / 工伤期间 / 医疗期内被辞退

### 必问项
- **入职日期** → 用于计算经济补偿 / 赔偿金的工作年限
- **解除日期 / 通知日期** → 用于确定劳动关系终止时点和仲裁时效起算
- **解除形式**（口头 / 书面 / 微信 / 邮件 / 短信）→ 用于判断解除程序合法性
- **解除理由**（用人单位陈述的具体原因）→ 用于对照 §39 / §40 / §41 的合法性审查
- **用人单位是否支付了补偿 / 赔偿 / 代通知金** → 用于判断用人单位自认违约的程度
- **劳动者是否处于特殊保护期**（孕期、产期、哺乳期、工伤、医疗期、工作满 15 年且距退休不足 5 年）→ 用于判断是否触发 §42 不得解除情形
- **是否经过工会程序**（用人单位是否事先通知工会）→ 用于审查 §43 程序合规性
- **是否签收解除证明** → 用于确认 §50 文书义务履行情况

### 加分项
- 解除前 12 个月平均工资（精确到元）→ 直接决定赔偿基数
- 工资构成（基本工资 + 岗位 + 绩效 + 补贴 + 加班费）→ 区分应发与实发，影响计算
- 是否被要求签"自愿离职 / 协商一致解除"协议 → 判断 §36 协商解除的合法性
- 用人单位是否事后补充证据（如规章、考勤、检讨）→ 用于判断理由"事后变更"的可采性
- 单位经营状况（裁员可能性）→ 区分 §40 单方解除与 §41 经济性裁员

### 证据清单
- **必备**
  - 解除劳动合同通知书 / 离职证明 / 终止劳动关系证明 → 证明解除事实与理由 → 立即索取原件，复印件加盖骑缝章
  - 劳动合同（含续订合同）→ 证明劳动关系存续 → 索取原件并复印，复印件加盖单位骑缝章，原件存放于防火防潮处
- **重要**
  - 工资发放记录（银行流水近 24 个月）→ 用于赔偿金 / 经济补偿基数计算 → 持身份证到银行打印
  - 用人单位陈述解除理由的微信 / 邮件 / 录音 → 用于锁定解除时点的理由，防止事后变更 → 录音务必保留原始载体（手机）
  - 考勤记录 / 排班表 → 用于反驳"严重违反规章制度"主张 → 申请劳动监察调取或自行截屏
- **加分**
  - 单位规章制度文本（如有签收）→ 证明规章未经民主程序或未公示
  - 同事证言 → 用于印证解除事实与理由
  - 工会沟通记录 → 用于证明 §43 程序违法

### 法条映射
- 主要依据：**劳动合同法 §39（用人单位单方解除的法定情形）**、**§40（无过失性辞退的提前通知义务）**、**§41（经济性裁员）**、**§42（不得解除的特殊情形）**、**§43（工会程序）**、**§48（违法解除的法律后果：继续履行或支付赔偿金）**、**§87（违法解除按经济补偿标准二倍支付赔偿金）**
- 配套：劳动法 §25-§29（解除的法定情形与禁止情形）、§98（违法解除的法律责任）
- 特殊：劳动合同法实施条例 §19-§21（解除条件细则，需 WebSearch 验证最新版）；最高法《关于审理劳动争议案件适用法律问题的解释（一）》（2020）相关条款

## 类型 2：拖欠工资 / 克扣工资

### 识别关键词
- "拖欠工资"、"欠薪"、"克扣工资"、"扣工资"、"罚工资"
- "工资没发"、"上个月工资还没到账"、"发了但少发了"
- "绩效不发了"、"奖金没了"、"提成不发"
- "工资分两次发"、"部分走对公部分走个人"、"工资发现金"
- "工资条对不上"、"工资少了 X 元"
- "试用期工资打 8 折"、"扣了培训费"、"扣了服装费"
- "公司说效益不好先欠着"、"打了欠条"

### 必问项
- **入职日期** → 用于确定应发工资的起算期间
- **正常工资标准**（合同约定 / 实际发放基数）→ 用于计算差额
- **欠薪期间**（起止月份）→ 用于精确计算应付金额
- **每月实发金额** → 用于对比应发与实发的差额
- **工资发放周期与方式**（次月 X 日、银行转账 / 现金 / 微信）→ 用于判断是否违反 §50 按月足额支付
- **工资构成**（基本 + 绩效 + 岗位 + 补贴 + 加班费）→ 用于区分应发与实发
- **是否有工资条 / 工资明细** → 用于举证克扣事实
- **是否在试用期内** → 用于审查试用期工资是否符合 §20（不低于本单位同岗位最低档工资 80% 或合同约定工资 80%，且不低于当地最低工资标准）

### 加分项
- 用人单位是否出具过欠条 / 承诺书 → 直接锁定债务
- 用人单位是否有书面扣款说明（罚款、培训费、服装费、绩效扣减）→ 用于审查扣款合法性
- 是否已离职（已离职可主张 §85 加付赔偿金 50%-100%）→ 影响是否申请劳动监察前置
- 是否低于当地最低工资标准 → 触发 §48 + §85 加付赔偿金
- 是否属于农民工 / 工程建设领域 → 适用《保障农民工工资支付条例》特别程序

### 证据清单
- **必备**
  - 劳动合同（约定工资标准）→ 证明应发金额 → 索取原件复印，原件与复印件分开存放，电子扫描件云端备份
  - 银行流水（工资入账记录，近 24 个月）→ 证明实发金额与发放周期 → 持身份证到银行打印并加盖银行业务公章
- **重要**
  - 工资条 / 工资明细 / 个税 APP 收入记录 → 用于比对应发与实发 → 个税 APP 截图保存
  - 考勤记录（与工资对应的出勤天数）→ 用于证明已提供劳动 → 截屏保存并导出原始数据文件，注明数据来源与抓取时间
  - 用人单位出具的欠条 / 承诺书 / 结算单 → 直接证明欠薪金额 → 要求加盖单位公章或法定代表人签字，原件留存于防火防潮处
- **加分**
  - 与 HR / 财务 / 老板的微信 / 邮件沟通记录 → 用于印证欠薪事实与原因
  - 同岗位同事的工资证明 → 用于审查是否同工同酬
  - 当地最低工资标准文件 → 用于审查 §48 最低工资保障

### 法条映射
- 主要依据：**劳动法 §50（工资应当以货币形式按月支付，不得克扣或者无故拖欠）**、**§48（最低工资保障制度）**、**§91（克扣、拖欠工资由劳动行政部门责令支付并加付赔偿金）**；**劳动合同法 §30（及时足额支付劳动报酬，劳动者可申请支付令）**、**§85（逾期不支付按应付金额 50%-100% 加付赔偿金）**
- 配套：劳动合同法 §20（试用期工资底线）；劳动法 §51（法定休假 / 婚丧假工资照发）
- 特殊：《保障农民工工资支付条例》（2020）；《工资支付暂行规定》（需 WebSearch 验证最新版）；地方工资支付条例（如《上海市企业工资支付办法》）

## 类型 3：未签劳动合同（二倍工资差额）

### 识别关键词
- "没签合同"、"没签劳动合同"、"至今没签书面合同"
- "入职 X 个月了还没签"、"试用期过了才签"
- "签了空白合同"、"合同被收走了"、"没给我一份"
- "签的是劳务合同 / 协议 / 合作协议"（实际为劳动关系）
- "用电子合同"、"在我手机上点的确认"
- "公司是另一家主体签的"（关联公司、劳务派遣、外包）
- "我从来没见过合同"

### 必问项
- **入职日期** → 用于计算二倍工资的起止期间（用工之日起满 1 个月的次日至满 1 年的前一日）
- **是否签过任何书面合同 / 协议** → 用于判断 §82 第 1 款适用前提
- **签订日期**（如签过）→ 用于计算未签期间
- **是否拿到合同文本** → 用于审查 §16 文本交付义务
- **劳动关系建立的方式**（直接入职 / 劳务派遣 / 外包 / 关联公司）→ 用于确定实际用人单位
- **入职后满 1 年是否仍未签** → 用于判断是否转化为 §14 视为订立无固定期限劳动合同
- **实际工资标准** → 用于计算二倍工资差额的基数
- **是否签过其他协议**（劳务、合作、实习）→ 用于识别"以非劳动关系协议规避劳动合同"的情形

### 加分项
- 是否有微信 / 邮件提及"合同还没签"或"等签合同"的记录 → 印证未签事实
- 入职前招聘信息（HR / 招聘平台截图）→ 证明用工意图
- 工作牌 / 工服 / 名片 / 邮箱前缀 → 证明事实劳动关系存在
- 是否存在关联公司混同用工 → 可能触发连带责任
- 是否在仲裁时效内（一般 1 年，自知道或应当知道权利被侵害之日起算）→ 影响胜诉

### 证据清单
- **必备**
  - 工资发放记录（银行流水）→ 证明事实劳动关系存在 → 持身份证到银行打印近 24 个月流水并加盖银行业务公章
  - 工作证 / 工牌 / 入职登记表 / 录用通知书 → 证明用工事实 → 拍照存档，原件索取并妥善保管；电子录用通知保存邮件原文与时间戳
- **重要**
  - 考勤记录 / 排班表 / 工作日志 → 证明接受用人单位管理 → 截屏保存并保留原始数据导出文件（钉钉 / 飞书可申请导出）
  - 工作服 / 名片 / 公司邮箱 → 证明身份从属性 → 拍照存档；公司邮箱收发邮件保留完整邮件头与原始账户登录记录
  - 微信 / 钉钉 / 飞书工作沟通记录 → 证明人格从属性与经济从属性 → 通过"腾讯电子签"小程序或公证处存证；切勿删除原始聊天记录与手机载体
- **加分**
  - 与 HR 沟通"合同何时签"的聊天记录 → 印证未签原因在用人单位
  - 同事证言 → 用于证明未签事实
  - 招聘平台 / 招聘信息截图 → 用于证明入职途径

### 法条映射
- 主要依据：**劳动合同法 §10（建立劳动关系应当订立书面合同；满 1 个月未订立的应立即补订）**、**§82（自用工之日起满 1 个月不满 1 年未订立书面合同，应每月支付二倍工资；应当订立无固定期限合同而未订立的，自应当订立之日起每月支付二倍工资）**、**§14（满 1 年未订立书面合同视为已订立无固定期限劳动合同）**、**§16（合同文本应由双方各执一份）**
- 配套：劳动法 §16（建立劳动关系应当订立劳动合同）、§19（劳动合同应当以书面形式订立）、§98（用人单位故意拖延不订立劳动合同的法律责任）
- 特殊：劳动合同法实施条例 §5-§7（未签书面合同的具体规则，需 WebSearch 验证最新版）

## 类型 4：加班费争议

### 识别关键词
- "加班"、"加班费"、"加班工资"、"延时加班"
- "周末加班"、"双休日加班"、"休息日加班"
- "法定节假日加班"、"国庆加班"、"春节加班"
- "996"、"007"、"早 9 晚 9"
- "加班没给钱"、"加班只给调休"、"加班算成调休"
- "加班不审批"、"加班要走流程"
- "下班后还在公司"、"周末被叫去上班"
- "考勤但不算加班"、"打卡不算加班"

### 必问项
- **工时制度**（标准工时 / 综合计算工时 / 不定时）→ 用于判断加班认定标准，综合计算工时需经审批
- **正常工作时间**（每日 / 每周）→ 用于判断是否超出法定上限
- **加班的具体时段**（工作日延时 / 休息日 / 法定节假日）→ 用于对应劳动法 §44 三档不同加班费率（150% / 200% / 300%）
- **加班时长**（每日具体小时数）→ 用于计算加班费金额
- **加班的频率**（每月多少天）→ 用于核算总额
- **是否经过审批** → 用于审查劳动合同法 §31 强迫加班禁止与劳动法 §41 加班协商程序
- **月工资基数** → 用于计算加班费计算基数（一般以正常出勤月工资为准，不含加班费本身）
- **是否有加班费支付记录 / 调休安排** → 用于判断用人单位是否已部分支付

### 加分项
- 是否存在"996 / 007"等制度化加班 → 可主张系统性违法
- 加班审批制度是否流于形式 → 实际加班但未走流程的，仍应认定加班
- 是否存在值班与加班混淆 → 值班与加班法律性质不同
- 是否已离职（离职后可一并主张，并触发 §85 加付赔偿金）
- 加班费计算基数是否被人为压低（如按基本工资、最低工资计算）→ 实务中应以正常出勤月工资为准

### 证据清单
- **必备**
  - 考勤记录（打卡、指纹、人脸识别、钉钉 / 飞书 / 企业微信打卡）→ 证明加班事实 → 立即导出原始数据文件并截屏（含时间戳与界面 URL），通过"腾讯电子签"或第三方存证平台固化
  - 工资发放记录 / 工资条 → 用于证明是否已支付加班费及支付标准 → 持身份证到银行打印流水并加盖业务公章；工资条扫描存档
- **重要**
  - 加班审批单 / 工作安排邮件 / 微信通知 → 证明用人单位安排加班（区别于自愿加班）→ 邮件保留原始 .eml 文件（含邮件头）；微信通知截屏并保留原始载体
  - 工作日志 / 工作交接记录 → 印证加班内容与时长 → 公司系统导出 PDF 或截屏（含账号信息与时间戳）；纸质日志拍照存档
  - 监控录像 / 门禁记录 → 用于补充证明在场时间 → 离职前向物业 / 安保部门申请调取并刻录光盘，或录像截屏存证
- **加分**
  - 同事证言 / 加班群聊天记录 → 印证加班制度化
  - 公司加班制度 / 规章 → 用于审查是否变相强迫加班
  - 加班期间的客户沟通记录（邮件、微信时间戳）→ 用于证明实际工作时间

### 法条映射
- 主要依据：**劳动法 §36（每日不超过 8 小时、每周不超过 44 小时）**、**§38（每周至少休息 1 日）**、**§41（加班一般每日不超过 1 小时，特殊原因不超过 3 小时，每月不超过 36 小时）**、**§43（不得违法延长工作时间）**、**§44（加班费标准：延时 150%、休息日 200%、法定节假日 300%）**；**劳动合同法 §31（不得强迫或变相强迫加班，安排加班应支付加班费）**、**§85（不支付加班费由劳动行政部门责令限期支付，逾期加付 50%-100% 赔偿金）**

- 特殊：《全国年节及纪念日放假办法》（法定节假日）；《国务院关于职工工作时间的规定》；地方工时审批办法（需 WebSearch 验证最新版）

## 类型 5：工伤认定与待遇

> **⚠️ 主法不在内置 references/ 中：** 本类争议的主要依据《工伤保险条例》未内置，
> 所有相关条文必须通过 WebSearch 实时检索。SKILL.md 在生成意见书时，
> 工伤保险条例相关引用必须标注 URL 与发布日期。

### 识别关键词
- "工伤"、"受了工伤"、"工伤认定"
- "上班时受伤"、"工作期间受伤"、"工作场所受伤"

- "公司没缴工伤保险"、"公司没报工伤"

### 必问项
- **事故发生日期与具体时间** → 用于判断是否在 30 日申报期内（用人单位应在 30 日内申报，劳动者可在 1 年内自行申报）
- **事故发生地点与经过** → 用于对照《工伤保险条例》§14 认定工伤的情形

- **是否为上下班途中交通事故**（含责任认定）→ 用于对照《工伤保险条例》§14 第 6 项，需劳动者非主要责任
- **是否患有职业病**（含职业史、职业健康检查）→ 用于对照《工伤保险条例》§14 第 4 项
- **用人单位是否缴纳工伤保险** → 用于区分由基金支付还是由用人单位支付的项目
- **是否已做工伤认定 / 劳动能力鉴定** → 用于确定当前阶段与待遇计算依据
- **伤残等级**（如已鉴定）→ 用于确定一次性伤残补助金月数（十级 7 个月至一级 27 个月）

### 加分项
- 是否签订过"私了协议" → 用于审查协议效力（显著低于法定标准的可撤销）
- 用人单位是否垫付医疗费 → 影响后续结算
- 停工留薪期是否超过 12 个月（延长需鉴定确认）
- 护理依赖等级 → 影响生活护理费
- 是否死亡（涉及工亡补助金、供养亲属抚恤金）

### 证据清单
- **必备**
  - 医疗机构诊断证明 / 病历 / 出院小结 → 证明伤情与治疗 → 病案室复印并加盖医院病案专用章；就诊期间实时索取门诊病历盖章件
  - 工伤认定决定书 / 劳动能力鉴定结论书（如已取得）→ 证明工伤与伤残等级 → 复印多份并云端备份原件，决定书原件存放于防火防潮处
- **重要**
  - 事故现场证据（照片、视频、监控）→ 证明事故经过与原因 → 事故发生后立即拍摄（含时间戳与地理定位）；视频上传云盘并保留原始拍摄载体
  - 交警事故认定书（上下班途中交通事故情形）→ 证明非主要责任 → 向交警部门索取原件并复印留存，复印件加盖交警队公章
  - 考勤记录 / 工作安排记录 → 证明事故发生时为工作时间、因工作原因 → 当日排班表 / 派工单 / 工作群通知截屏，注明数据来源与抓取时间
  - 工伤保险缴费记录 → 证明用人单位是否依法参保 → 登录当地社保 APP 或国家社会保险公共服务平台下载 PDF 并截屏（含查询时间戳）
- **加分**
  - 现场目击证人证言 → 印证事故经过
  - 工资发放记录（受伤前 12 个月）→ 用于计算停工留薪期工资与一次性伤残补助金基数
  - 护理人员收入证明 → 用于计算护理费

### 法条映射
- 主要依据：**劳动法 §57（伤亡事故和职业病统计报告与处理制度）**、**§73（因工伤残或患职业病享受社会保险待遇）**、**§29（患职业病或因工负伤并被确认丧失或部分丧失劳动能力的，不得依据 §26、§27 解除劳动合同）**；**劳动合同法 §42（在本单位患职业病或因工负伤并被确认丧失或部分丧失劳动能力的，用人单位不得依据 §40、§41 解除劳动合同）**、**§45（丧失或部分丧失劳动能力劳动者的合同终止按工伤保险规定执行）**
- 配套：劳动法 §70-§74（社会保险制度与基金管理）、§92（劳动安全设施与劳动卫生不符合规定的法律责任）、§100（不缴纳社会保险费的法律责任）
- 特殊：**《工伤保险条例》（国务院令第 586 号，2010 修订）**为工伤认定与待遇的核心法规（需 WebSearch 验证最新版）；地方工伤保险实施办法；**《职业病防治法》**

## 类型 6：调岗降薪 / 变更劳动合同

### 识别关键词
- "调岗"、"调岗降薪"、"降薪"、"降工资"、"调薪"
- "调到 XX 岗位"、"换了部门"、"换工位"
- "调到外地"、"调到其他城市"、"调到分公司"
- "把工资降了"、"绩效下调"、"基本工资降了"
- "不签调岗同意书就…"、"不接受就…"
- "说我不能胜任"、"说我能力不行所以调岗"
- "突然通知调岗"、"昨天通知今天调"

### 必问项
- **原岗位 / 原工资标准** → 用于确定变更前基准
- **新岗位 / 新工资标准** → 用于确定变更后差异，量化损失
- **调岗通知日期与生效日期** → 用于判断是否给予合理过渡期
- **调岗通知的形式**（书面 / 口头 / 微信 / 邮件）→ 用于审查 §35 书面形式要件
- **是否经过协商一致** → 用于判断是否符合 §35 协商一致原则
- **用人单位陈述的调岗理由** → 用于对照 §40 第 2 项（不能胜任）等单方变更情形
- **新岗位与原岗位的差异**（地点、薪酬、职责、职级、专业对口性）→ 用于审查调岗的合理性（是否具有侮辱性、惩罚性、针对性）
- **劳动者是否已实际到新岗位工作** → 用于判断是否构成"实际履行超过 1 个月"视为接受变更

### 加分项
- 是否存在报复性调岗（因举报、维权、申请工伤等）→ 用于主张调岗无效
- 新岗位是否具有侮辱性 / 惩罚性 → 法院倾向认定无效
- 用人单位是否在调岗同时威胁解除 → 可能构成被迫离职，触发 §38 + §46 经济补偿
- 是否处于特殊保护期（孕期、产期、哺乳期、医疗期）→ 调岗需额外合规
- 调岗是否经过民主程序（如涉及薪酬制度变更）→ 触发劳动合同法 §4 程序要求

### 证据清单
- **必备**
  - 劳动合同（含岗位、薪酬条款）→ 证明原约定 → 索取原件复印并加盖骑缝章，电子扫描件云端备份
  - 调岗 / 调薪通知书 → 证明变更事实与单方性质 → 索取书面原件；电子送达的保留系统记录与短信 / 邮件原文（含发件人、时间戳）
- **重要**
  - 工资发放记录（调岗前后）→ 证明降薪幅度 → 持身份证到银行打印调岗前后各 6 个月流水并加盖业务公章
  - 用人单位陈述调岗理由的微信 / 邮件 / 录音 → 用于锁定理由，防止事后变更 → 录音务必保留原始手机载体；微信 / 邮件通过"腾讯电子签"小程序或公证处存证
  - 原岗位与新岗位的工作内容证据（职位说明书、邮件签名、组织架构）→ 用于审查合理性 → 截屏保存（含时间戳、账号信息），导出原始文件；纸质文件拍照存档
- **加分**
  - 考勤记录（含两个岗位的工作地点）→ 用于证明岗位实际差异
  - 同事证言 / 内部公告 → 印证调岗的针对性与侮辱性
  - 用人单位规章制度（含调岗制度）→ 用于审查制度是否经过劳动合同法 §4 民主程序

### 法条映射
- 主要依据：**劳动合同法 §29（用人单位与劳动者应按合同约定全面履行各自义务）**、**§35（变更劳动合同应当协商一致并采用书面形式；变更后的合同文本由双方各执一份）**、**§38（用人单位未按约定提供劳动保护或劳动条件、未足额支付劳动报酬的，劳动者可解除合同并主张经济补偿）**、**§40 第 2 项（劳动者不能胜任工作，经培训或调岗仍不能胜任的，用人单位可提前 30 日通知或支付代通知金后解除）**
- 配套：劳动法 §17（订立和变更劳动合同应协商一致）、§26 第 2 项（劳动者不能胜任工作的解除情形）
- 特殊：最高法《关于审理劳动争议案件适用法律问题的解释（一）》关于调岗合理性的裁判规则（需 WebSearch 验证最新版）；地方裁审指引（如《上海市高级人民法院关于审理劳动争议案件若干问题的意见》）

### 必问项
- **是否签过竞业限制协议 / 保密协议 / 含竞业条款的劳动合同** → 用于确定协议效力
- **签订日期与协议期限** → 用于审查是否超过 §24 规定的 2 年上限
- **协议约定的竞业限制范围**（行业、地域、竞争对手名单）→ 用于审查 §24 合理性
- **协议约定的经济补偿标准与支付方式** → 用于审查是否符合"按月支付"要件
- **离职日期** → 用于起算竞业限制期限与经济补偿支付期
- **离职后用人单位是否实际按月支付经济补偿** → 用于审查用人单位是否履行义务
- **是否已入职新单位 / 新单位是否与原单位有竞争关系** → 用于判断是否构成违约
- **劳动者岗位是否属于 §24 适用对象**（高级管理人员 / 高级技术人员 / 其他负有保密义务的人员）→ 用于审查协议对主体的适用性

### 加分项
- 用人单位连续 3 个月未支付经济补偿 → 劳动者可请求解除竞业限制约定（最高法司法解释四第 8 条）
- 协议未约定经济补偿标准 → 适用司法解释：劳动者离职前 12 个月平均工资的 30%，且不低于当地最低工资标准
- 用人单位将竞业补偿"提前纳入工资" → 实务中可能被认定无效
- 是否属于"被动违约"（如新单位隐瞒竞争关系）→ 影响违约责任分担
- 用人单位是否在离职时明确通知"无需履行竞业限制" → 劳动者可主张额外 3 个月补偿

### 证据清单
- **必备**
  - 竞业限制协议 / 保密协议 / 含竞业条款的劳动合同 → 证明约定内容 → 索取签字盖章原件，复印并云端备份；如系电子合同，保留完整签署流程记录与哈希值
  - 离职证明 / 解除或终止劳动合同通知书 → 证明离职时点（竞业期限起算）→ 索取原件并加盖单位公章，复印件分开存放
- **重要**
  - 用人单位支付竞业经济补偿的银行流水 → 证明用人单位是否按月足额支付 → 持身份证到银行打印离职后逐月流水并加盖业务公章，标注每月到账项
  - 原单位与新单位的经营范围对比（工商信息）→ 用于判断是否构成"竞争关系" → 通过"国家企业信用信息公示系统"截图保存（含查询时间戳），导出工商档案 PDF
  - 新单位的劳动合同 / 入职证明 → 证明新就业情况 → 索取签字盖章原件并复印留存，电子扫描件云端备份
- **加分**
  - 用人单位通知"无需履行竞业限制"的书面文件 → 用于主张额外 3 个月补偿
  - 用人单位主张违约的证据（监控、调查报告）→ 用于审查证据合法性
  - 行业惯例 / 同业名单公开资料 → 用于审查竞业范围的合理性

### 法条映射
- 主要依据：**劳动合同法 §23（约定保守商业秘密与知识产权保密事项；可约定竞业限制条款，按月支付经济补偿；劳动者违反的应支付违约金）**、**§24（竞业限制的人员限于高级管理人员、高级技术人员和其他负有保密义务的人员；竞业限制期限不得超过 2 年）**、**§25（除 §22 服务期和 §23 竞业限制外，用人单位不得与劳动者约定由劳动者承担违约金）**
- 配套：劳动合同法 §90（劳动者违反保密义务或竞业限制的赔偿责任）
- 特殊：原《劳动争议司法解释（四）》（法释〔2013〕4 号）第 6-10 条已于 2021-01-01 起被新《劳动争议司法解释（一）》（法释〔2020〕26 号）整合取代。原竞业限制规则（经济补偿标准、3 个月未支付可解除、用人单位解除需额外支付 3 个月补偿等）现分布于新解释（一）中，**具体条款编号需 WebSearch 验证现行版本**

## 类型 8：服务期与违约金（培训服务期）

### 识别关键词
- "服务期"、"培训服务期"、"服务期协议"
- "违约金"、"培训费违约金"
- "公司出钱培训了我"、"送我去培训"

- "专项培训"、"专业技术培训"

### 必问项
- **是否签过服务期协议 / 含服务期条款的合同** → 用于确定协议效力
- **培训性质**（专业技术培训 / 普通岗前培训 / 安全培训 / 业余学习）→ 用于审查是否符合 §22"专项培训费用 + 专业技术培训"双要件，普通培训不得约定服务期
- **用人单位实际支付的培训费用**（含发票、票据、汇款记录）→ 用于确定违约金上限
- **约定的服务期年限** → 用于审查合理性
- **已经履行的服务期**（入职日期 / 培训完成日期 / 离职日期）→ 用于按比例递减计算
- **离职日期与离职原因**（劳动者主动 / 协商 / 用人单位违法解除）→ 用于审查违约金责任分担
- **违约金的具体数额** → 用于审查是否超过 §22 上限
- **是否同时被要求返还培训费与支付违约金** → 用于审查是否违反"二者择一"原则

### 加分项
- 培训是否包含学历教育（MBA、博士）→ 服务期与违约金规则可能不同
- 培训费用是否包含工资、差旅、住宿 → 实务中对"培训费用"范围有争议
- 用人单位违法解除导致劳动者无法继续履行服务期 → 劳动者无须支付违约金
- 是否约定了双重违约金（培训服务期 + 竞业限制）→ 需分别审查
- 是否属于"户口服务期" → 不适用 §22，按地方政策处理

### 证据清单
- **必备**
  - 服务期协议 / 培训协议 / 含服务期条款的合同 → 证明约定内容 → 索取双方签字盖章原件，复印并云端备份；电子合同保留完整签署记录
  - 培训费用发票 / 票据 / 汇款记录 → 用于确定违约金上限 → 增值税发票应通过"国家税务总局全国增值税发票查验平台"核验并截图存档；汇款记录打印银行电子回单
- **重要**
  - 培训机构的资质证明 / 培训课程证明 → 用于审查是否为"专业技术培训" → 索取培训机构资质证书复印件并加盖机构公章，课程大纲 / 学员名单拍照存档
  - 培训完成证明 / 结业证书 → 用于确定服务期起算时点 → 索取原件并复印，电子扫描件云端备份
  - 劳动者离职证明 / 解除劳动合同通知书 → 用于确定已履行服务期 → 索取原件并加盖单位公章，复印件分开存放
- **加分**
  - 用人单位培训制度 / 规章 → 用于审查培训性质
  - 同期参训同事证言 → 印证培训实际内容
  - 用人单位主张违约金的计算明细 → 用于审查是否超过法定上限

### 法条映射
- 主要依据：**劳动合同法 §22（用人单位提供专项培训费用进行专业技术培训的，可约定服务期；劳动者违反服务期约定的应支付违约金；违约金不得超过培训费用；不得超过服务期尚未履行部分所应分摊的培训费用）**、**§25（除 §22 服务期与 §23 竞业限制外，用人单位不得与劳动者约定由劳动者承担违约金）**
- 配套：劳动合同法 §90（劳动者违法解除劳动合同或违反约定的赔偿责任）
- 特殊：劳动合同法实施条例 §26（服务期与劳动合同期限的关系，需 WebSearch 验证最新版）；地方关于户口服务期的政策（如北京、上海的相关裁审指引）

## 类型 9：经济补偿金 / 赔偿金计算

### 识别关键词
- "经济补偿"、"经济补偿金"、"补偿金"
- "赔偿金"、"违法解除赔偿金"、"2N"
- "N"、"N+1"、"2N"
- "代通知金"、"一个月工资代替通知"
- "解除补偿"、"离职补偿"
- "协商解除补偿"
- "工作 X 年应该赔多少"

### 必问项
- **入职日期与离职日期** → 用于计算工作年限（满 1 年算 1 个月，满 6 个月不满 1 年按 1 年，不满 6 个月按半个月）
- **离职前 12 个月平均工资** → 用于确定补偿 / 赔偿基数（应发工资，含奖金、津贴、补贴）
- **离职原因** → 用于区分是否触发 §46 应当支付经济补偿的情形，以及是否构成 §48 + §87 违法解除赔偿金
- **用人单位是否已支付补偿 / 赔偿** → 用于计算差额
- **是否属于高收入劳动者**（月工资是否超过当地上年度职工月平均工资 3 倍）→ 用于触发 §47 第 2 款"封顶"规则（按 3 倍计算，年限最高 12 年）
- **工资构成**（基本工资 / 岗位 / 绩效 / 补贴 / 加班费 / 年终奖）→ 用于审查基数是否被人为压低
- **是否触发 §42 不得解除情形** → 用于判断是否构成违法解除（2N）
- **是否经过协商解除及协商提出方** → §46 第 2 项：用人单位提出协商解除的应支付经济补偿，劳动者提出的无需支付

### 加分项
- 是否有未休年休假工资可一并主张 → 按劳动法 §44 法定节假日 300% 计算
- 是否有未结清的加班费 / 工资 → 可一并主张，并触发 §85 加付赔偿金
- 是否属于"合同到期不续签"情形 → §46 第 5 项区分用人单位维持 / 提高条件 vs 降低条件
- 是否触发"视为订立无固定期限合同"情形 → 影响赔偿性质
- 用人单位是否在解除时已自认支付 N 或 N+1 → 用于主张 2N 差额

### 证据清单
- **必备**
  - 劳动合同（含续订合同）→ 证明劳动关系与起止时间 → 索取原件复印加盖骑缝章，原件与复印件分开存放，电子扫描件云端备份
  - 解除 / 终止劳动合同通知书 → 证明解除事实与理由 → 立即索取原件，复印件加盖单位骑缝章；电子送达的保留系统记录与短信 / 邮件原文
  - 工资发放记录（离职前 12 个月）→ 用于计算补偿 / 赔偿基数 → 持身份证到银行打印离职前 12 个月流水并加盖业务公章
- **重要**
  - 工资条 / 个税 APP 收入记录 → 用于核实工资构成 → 个税 APP 截图保存（含登录账号与时间戳），纸质工资条扫描存档
  - 用人单位支付补偿 / 赔偿的银行流水（如已部分支付）→ 用于计算差额 → 持身份证到银行打印并加盖业务公章，标注每一笔到账的性质（补偿 / 工资）
  - 离职交接单 / 离职证明 → 用于锁定离职时点与原因 → 索取原件并加盖单位公章，复印件分开存放；电子扫描件云端备份
- **加分**
  - 当地上年度职工月平均工资数据 → 用于审查是否触发 §47 封顶规则
  - 用人单位内部补偿方案 / 会议纪要 → 用于印证补偿标准
  - 同岗位同事的赔偿案例 → 用于协商谈判参考

### 法条映射
- 主要依据：**劳动合同法 §46（应当支付经济补偿的 7 种法定情形）**、**§47（经济补偿按工作年限计算：满 1 年 1 个月工资；6 个月以上不满 1 年按 1 年；不满 6 个月半个月工资；高收入劳动者按当地职工月平均工资 3 倍封顶、年限最高 12 年；月工资为解除或终止前 12 个月平均工资）**、**§48（违法解除或终止的法律后果：继续履行或按 §87 支付赔偿金）**、**§87（违法解除或终止按 §47 经济补偿标准的 2 倍支付赔偿金）**
- 配套：劳动法 §28（依 §24 协商、§26 无过失性辞退、§27 经济性裁员解除的应给予经济补偿）；劳动合同法 §97（本法施行前后经济补偿年限的计算衔接）
- 特殊：劳动合同法实施条例 §10、§22-§25（经济补偿具体规则，需 WebSearch 验证最新版）；最高法相关司法解释关于 §47 月工资计算的细则

## 类型 10：社保与公积金争议

> **⚠️ 部分主法不在内置 references/ 中：** 本类争议的主要依据《社会保险法》
> 与《住房公积金管理条例》未内置，相关条文必须通过 WebSearch 实时检索。
> 注意：公积金争议不属于劳动仲裁受案范围，需通过公积金管理中心投诉。

### 识别关键词
- "社保"、"社会保险"、"五险一金"、"没缴社保"
- "养老保险"、"医疗保险"、"失业保险"、"工伤保险"、"生育保险"
- "公积金"、"住房公积金"
- "没给我交社保"、"断缴"、"漏缴"
- "试用期不缴社保"（违法）
- "工资发现金不缴社保"（违法）
- "签了放弃社保协议"（约定无效）
- "社保基数低了"、"按最低基数缴"
- "想补缴社保"、"想补缴公积金"

### 必问项
- **入职日期** → 用于确定应缴社保的起算期间
- **离职日期 / 当前是否在职** → 用于确定补缴期间终点
- **用人单位是否办理社保登记** → 用于审查 §72 强制参保义务
- **缴费基数**（实际缴费基数 vs 应发工资）→ 用于计算差额与补缴金额
- **缴费险种**（是否五险全缴）→ 用于审查是否漏缴某一险种
- **是否签过"放弃社保协议"或"现金折现协议"** → 用于说明该类约定依 §72 + 社会保险法无效
- **是否已发生社保待遇损失**（如医疗费不能报销、生育津贴不能领取、养老金减少）→ 用于主张赔偿损失
- **是否已离职**（在职期间通常需通过劳动监察 / 社保征收机构投诉，离职后可在仲裁中一并主张损失赔偿）→ 用于选择救济路径

### 加分项
- 是否存在断缴影响买房 / 买车 / 落户 / 子女入学资格 → 可主张实际损失赔偿
- 是否有因未缴工伤保险导致的工伤待遇损失 → 用人单位需按工伤保险条例标准全额承担
- 是否有因未缴生育保险导致的生育津贴损失 → 用人单位需赔偿
- 是否有因未缴医疗保险导致的医疗费不能报销损失 → 用人单位需赔偿
- 公积金是否缴存（公积金争议由公积金管理中心处理，非劳动仲裁范围）→ 用于区分救济路径

### 证据清单
- **必备**
  - 劳动合同 → 证明劳动关系与应缴起算时点 → 索取原件复印加盖骑缝章，原件与复印件分开存放，电子扫描件云端备份
  - 社保缴费记录（可登录当地社保 APP / 国家社会保险公共服务平台查询打印）→ 证明缴费情况 → 登录平台下载 PDF 明细并截屏（含登录账号与查询时间戳），多次打印分时段留存
- **重要**
  - 工资发放记录 → 用于核算应缴基数（一般以应发工资为准）→ 持身份证到银行打印并加盖业务公章
  - 个税 APP 收入记录 → 用于印证实际工资 → 个税 APP 截图保存（含登录账号与时间戳），导出收入明细
  - "放弃社保协议" / "现金折现协议"（如有）→ 用于证明用人单位违法主观状态 → 索取原件或拍照存档，邮件 / 微信发送的保留原始载体与时间戳，必要时公证
- **加分**
  - 社保待遇损失的证据（医疗费发票、生育医疗费发票、养老金核算单）→ 用于主张损失赔偿
  - 用人单位未办社保登记的证据 → 用于劳动监察投诉
  - 公积金账户查询记录 → 用于公积金投诉

### 法条映射
- 主要依据：**劳动法 §72（用人单位和劳动者必须依法参加社会保险，缴纳社会保险费用）**、**§73（劳动者在退休、患病负伤、因工伤残或患职业病、失业、生育等情形下依法享受社会保险待遇）**、**§100（用人单位无故不缴纳社会保险费的，由劳动行政部门责令限期缴纳，逾期可加收滞纳金）**；**劳动合同法 §38 第 3 项（用人单位未依法为劳动者缴纳社会保险费的，劳动者可解除合同并主张经济补偿）**、**§17 第 1 款第 7 项（社会保险为劳动合同必备条款）**
- 配套：劳动法 §70-§71、§74（社会保险制度与基金管理）；劳动合同法 §74 第 6 项（劳动监察对用人单位参加社会保险和缴费情况的监督检查）
- 特殊：**《中华人民共和国社会保险法》（2010 通过，2018 修正）**第 4 条、第 10 条、第 57 条、第 58 条、第 60 条、第 61 条、第 62 条、第 63 条、第 84 条、第 86 条等（用人单位登记、申报、缴费义务及滞纳金、罚款规则）；**《住房公积金管理条例》**第 13-§15、§20、§37-§38（公积金登记、缴存义务与罚则，需 WebSearch 验证最新版）
