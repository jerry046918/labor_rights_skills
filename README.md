# Labor Rights Advisor Skill

为面对中国大陆劳动争议的员工提供结构化法律意见的 Claude Code skill。

## 安装

将本目录放入 Claude Code skills 路径，或通过 plugin 机制加载。

## 使用

在 Claude Code 中描述您的劳动争议即可自动激活。例如：

> "我在上海某公司工作了 3 年，上周被以业绩不达标为由口头辞退，没给补偿，怎么办？"

skill 会引导您完成 5 阶段流程并生成《法律意见书》。

## 重要声明

本 skill 生成的意见书**不构成正式法律意见**。涉及重大权益（大额索赔、刑事责任、
竞业限制等）请咨询执业律师。
